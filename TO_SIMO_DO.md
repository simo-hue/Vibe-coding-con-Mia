# Azioni Manuali per Simo

- [ ] **Risolvere Problema di Rete**: Il terminale non riesce a risolvere `github.com`. 
  - Prova a controllare la tua connessione internet nel terminale con `ping github.com`.
  - Se il browser funziona ma il terminale no, potrebbe essere un problema di DNS del terminale o di un proxy/VPN attivo.
- [ ] **Eseguire il Push**: Una volta risolto il problema di rete, esegui:
  ```bash
  git push -u origin main
  ```
